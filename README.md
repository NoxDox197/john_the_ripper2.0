# john_the_ripper2.0 (Work in Progress)
⚠️ Questo progetto è ancora in fase di sviluppo.

## Descrizione

Script Python che automatizza l’uso di John the Ripper su WSL per il cracking di file protetti (ZIP, PDF, RAR, ecc.).

L’obiettivo è semplificare:

* estrazione hash
* scelta modalità di attacco
* esecuzione di John

## Requisiti

* Windows
* WSL installato
* John the Ripper (jumbo) configurato in:
  ~/john-jumbo/run

## Come usare

1. Avvia lo script Python
2. Inserisci il file da analizzare
3. Scegli il tipo di file
4. Rispondi alle domande sulla password (opzionale)

## Note
